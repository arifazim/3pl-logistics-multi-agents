Autonomous Loop Engineering

Three bounded self-improvement loops with deterministic guardrails and maximum iteration counts

1
Vendor Evaluator Loop
MCP: rate_card + vendor · Max 5 iterations
1. MCP ranks all vendors (70% reliability / 30% cost)
2. Pick next untried candidate
3. Compute margin deterministically (Python only)
4. If margin ≥ 12% + reliability ≥ 85% → select
5. Else: mark tried, shrink set, continue
🛡 Max 5 iterations · Tried-set prevents cycling · HITL if margin gap > 2%

Compliance-Critic Loop
MCP: policy · Max 3 iterations
1. Generate initial assignment plan
2. Check compliance: margin / SLA / weight
3. If violations: replan (next vendor / express / split)
4. Repeat max 3 rounds
5. Escalate to HITL if still violating

Agentic Guardrails

Four protection layers preventing bad automation decisions before they reach customers

Margin Protection

Deterministic · QuotationEngine

Rule: customer_price = vendor_cost ÷ (1 − 0.12) — 12% floor in Python, never LLM
Trigger: margin < 12% → Loop 1 tries next vendor → HITL if all fail
Proof: test_margin_is_floor_not_rate_card_tautology

Prompt Injection Defense

Security · VendorTextSanitizer

Runs first: every vendor text sanitized before entering agent context
Detects: ignore_instructions · system_override · role_override · jailbreak · delimiter_injection
If flagged: text redacted → HITL triggered immediately

HITL Escalation Gate

Threshold-based · Not an Agent

① Margin < 12% — low margin protection violation
② Load value ≥ $10,000 — high-value load
③ Compliance check failed (any of 3 policies)
④ Vendor text flagged by sanitizer
Routine loads dispatch automatically.

Trajectory Evaluator

8-step Gherkin assertions

① vendor_quotes_gathered · ② vendor_selected (score > 0)
③ customer_quote_computed (rate > 0) · ④ margin_from_selected_vendor
⑤ margin_protection ≥ 12% · ⑥ compliance_checked
⑦ hitl_gate_evaluated · ⑧ vendor_text_sanitized (conditional)
